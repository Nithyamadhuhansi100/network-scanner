# Network Scanner Tool

## Description
This is a simple Java networking project that scans IP ranges in a network and detects active devices.  
It shows whether a device is ONLINE or OFFLINE and optionally displays the hostname.

## Features
- Scan a range of IP addresses
- Detect online/offline devices
- Show hostname of active devices
- Easy to use console-based application

 ## How to Run
1. Open terminal in the project folder.
2. Compile the code:
```bash
javac NetworkScanner.java
